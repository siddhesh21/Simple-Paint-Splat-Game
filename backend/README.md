# Back-end part of the game

## Launch the Node JS server

- Install Node JS (v12.16.1 preferred)
- Run the following commands:

```$> npm install```
```$> npm start```


## Generate documentation

- Install jsdoc globally

```$> npm i jsdoc -g```

- Run jsdoc with the provided config file

```$> jsdoc -c jsdoc.json```
